# 에이벤처스 Claude Cowork 프롬프트팩

> 사용법: 실습 중 또는 후속 검토 때 필요한 프롬프트만 복사해 사용한다.  
> 원칙: 실제 민감자료가 아니라 더미 또는 비식별 자료로만 사용한다.

## 1. 공통 안전 프롬프트

```text
이 작업은 교육용 더미 자료 또는 비식별 자료를 기준으로 진행한다.
실제 IR Deck, 투자계약서, 텀시트, LP 자료, 포트폴리오 미공개 실적, 개인정보, 계정정보는 사용하지 않는다.
투자 여부, 기업가치, 법률·회계 판단을 확정하지 마.
근거가 부족한 내용은 "확인 필요"로 표시해줘.
AI가 도울 일과 사람이 최종 판단할 일을 분리해서 작성해줘.
```

## 2. IR Deck 1차 스크리닝

```text
첨부한 자료는 교육용 또는 비식별 IR 자료다.
투자 여부를 판단하지 말고, 투자심사역이 1차 검토할 수 있도록 정리해줘.

출력:
1. 회사 개요
2. 문제와 솔루션
3. 시장/고객/경쟁 구도
4. 사업모델과 현재 지표
5. 투자 매력 포인트
6. 핵심 리스크
7. 추가 확인자료 요청 목록
8. 파트너 미팅 질문 10개
9. 사람이 판단해야 할 항목

근거가 부족하면 "확인 필요"로 표시해줘.
```

## 3. IC 예상 질문 생성

```text
아래 투자검토 메모 초안을 기준으로 IC에서 나올 수 있는 질문을 만들어줘.

질문 유형:
- 시장성
- 제품/기술 차별성
- 팀 역량
- 매출 반복성
- 고객 집중도
- 경쟁사 대비 우위
- 투자 조건
- 후속 투자 가능성
- 회수 가능성
- 법률·회계 리스크

각 질문마다 왜 중요한지, 어떤 자료로 확인해야 하는지 함께 써줘.

[투자검토 메모 붙여넣기]
```

## 4. 산업 리서치 구조화

```text
아래 자료를 바탕으로 산업 리서치 메모를 작성해줘.

출력:
1. 산업 한 줄 정의
2. 밸류체인
3. 주요 고객군
4. 주요 플레이어와 경쟁 구도
5. 최근 기술 변화
6. 규제/정책 이슈
7. 투자 관점에서 볼 매력
8. 투자 관점에서 볼 리스크
9. 추가로 확인할 질문

자료에 없는 내용은 추정하지 말고 "추가 조사 필요"로 표시해줘.

[자료 붙여넣기]
```

## 5. 포트폴리오 월간 업데이트 요약

```text
아래 포트폴리오 월간 업데이트는 교육용 또는 비식별 자료다.
투자/사후관리 담당자가 빠르게 볼 수 있도록 요약해줘.

출력:
1. 이번 달 핵심 변화
2. 긍정 신호
3. 위험 신호
4. 런웨이/매출/비용/채용 관련 확인 필요
5. 후속 투자 또는 브릿지 검토가 필요한지에 대한 질문
6. 회사에 요청할 추가 자료
7. 파트너에게 보고할 5줄 요약

단정하지 말고 확인 질문 중심으로 작성해줘.

[월간 업데이트 붙여넣기]
```

## 6. LP 보고 초안

```text
아래 자료를 바탕으로 LP 보고 초안을 작성해줘.

주의:
- 실제 LP명, 출자금액, 수익률, 민감 수치는 사용하지 않는다.
- 수치가 없으면 임의로 만들지 않는다.
- 대외 보고 전 사람이 확인해야 할 항목을 따로 표시한다.

출력:
1. 보고 요약
2. 투자 집행 현황
3. 포트폴리오 주요 업데이트
4. 회수/후속 투자 관련 이슈
5. 확인 필요 수치
6. 대외 공유 전 검토 항목

[보고 재료 붙여넣기]
```

## 7. Legal/Risk Reviewer

```text
아래 투자검토 메모를 리스크 리뷰어 관점으로 검토해줘.

검토 기준:
- 법률 리스크
- 회계/재무 리스크
- 데이터 부족
- 과장 표현
- 투자 결론 단정
- 외부 공유 위험
- 추가 전문가 검토 필요 여부

출력 표:
| 항목 | 리스크 | 근거 | 추가 확인자료 | 사람 검토 필요 |

[메모 붙여넣기]
```

## 8. Cowork 작업 시작 프롬프트

```text
이 폴더는 교육용 투자검토 실습 폴더다.
먼저 파일 목록을 확인하고, 어떤 파일을 어떤 목적으로 사용할지 계획만 보여줘.
아직 파일을 수정하거나 새 파일을 만들지 마.

출력:
1. 이해한 목표
2. 사용할 파일
3. 실행 단계
4. 생성할 산출물
5. 내가 승인해야 할 지점
6. 민감정보 또는 외부 전송 위험
```

## 9. AI Native 업무 적용 아이디어 정리

```text
우리 회사의 투자검토 업무 중 MAX VCTeam으로 먼저 검증할 후보를 정리하려고 한다.

아래 업무 설명을 읽고 AI Native 방식으로 적용할 수 있는지 평가해줘.

출력:
1. 업무 요약
2. AI가 도울 수 있는 부분
3. 사람이 반드시 판단해야 할 부분
4. 필요한 비식별 샘플 자료
5. 사용하면 안 되는 민감자료
6. 2주 안에 측정할 수 있는 성공 기준

[업무 설명 붙여넣기]
```
