# 프로젝트 요청서 더미

> 교육용 더미 자료

## 요청명

신규 고객 온보딩 자료 개선

## 배경

신규 고객에게 전달되는 안내 자료가 여러 문서로 나뉘어 있고, 팀마다 사용하는 표현이 다르다. 고객지원팀은 반복 문의가 줄지 않는다고 느끼고 있으며, 영업팀은 고객에게 보낼 수 있는 짧은 설명 자료를 원한다.

## 현재 문제

- 고객 안내문, FAQ, 내부 답변 예시가 서로 다르다.
- 외부 발송 가능한 표현과 내부 참고용 표현이 구분되어 있지 않다.
- 고객이 다음 단계와 준비물을 자주 다시 묻는다.
- 기존 문서 중 최신 버전이 무엇인지 모호하다.

## 원하는 결과

- 신규 고객용 1페이지 안내문
- 내부 고객지원 FAQ
- 영업팀용 짧은 설명 문구
- 법무 또는 담당자 확인이 필요한 표현 목록

## 제약 조건

- 실제 고객명이나 계약 내용은 사용하지 않는다.
- 혜택, 가격, 법적 조건은 확정 문구로 작성하지 않는다.
- 외부 발송 전 담당자 검토가 필요하다.
- 2주 안에 초안을 만들고 내부 검토를 시작하고 싶다.

## 아직 정해지지 않은 것

- 최종 승인자
- 사용해야 할 최신 원문
- 외부 공개 가능한 고객 사례 범위
- FAQ의 담당 부서
