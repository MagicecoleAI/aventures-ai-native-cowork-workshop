## 운영 컨셉

이번 과정은 Claude 기능 소개가 아니라, 에이벤처스의 투자검토·리서치·보고 업무를 AI Native 팀이 어떻게 도울 수 있는지 체감하는 과정이다.

핵심 메시지는 다음 한 문장이다.

> VC의 AI 활용은 투자 판단을 자동화하는 것이 아니라, 투자검토·리서치·문서화·사후관리·리스크 검토를 더 빠르고 일관되게 수행하는 AI Native 팀을 만드는 일이다.

## 권장 준비

- 참가자 노트북
- Claude Desktop 설치
- Claude 유료 플랜 또는 조직 계정
- 첨부 PDF 기준 실습자료 다운로드
- Chrome 브라우저
- PlayMCP와 카카오톡 나챗방은 보안 정책에 따라 선택 실습 또는 강사 데모로 운영
- 실제 IR Deck, 투자계약서, LP 자료, 포트폴리오 미공개 실적을 쓰지 않는다는 공통 원칙

Cowork 접근이 어려운 참가자는 강사 데모를 보며 워크시트와 회고 중심으로 참여한다.
